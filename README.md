# Brute Force Protection System (Python)

This project simulates a login system protected against brute force attacks.

Python ile geliştirilmiş bir güvenlik sistemidir. 
Brute force saldırılarını engellemek için IP bazlı takip ve gecikme mekanizması kullanır.

## Features

- IP-based attempt tracking
- Automatic IP simulation
- Account blocking after multiple failed attempts
- Increasing delay system (anti-brute force)

## Usage

1. Run the script:
   python ip_login.py
2. Enter username and password
3. System simulates different IPs and blocks suspicious activity
